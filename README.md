# CASTLEBERRY BLOOM

**Mission-Locked Autonomous Agent Collective**

Live Demo: Open `index.html` in any browser

### Problem
AI agents drift without strong mission alignment. Enterprises lack a practical way to test and prove coherence at scale.

### Solution
Castleberry Bloom is a single-file browser platform with 7 live agents that demonstrate mission-locking in real time using Groq inference.

New enhancements:
- Audit-OR cross-border fraud detection
- Team A constitutional constraints
- Enhanced Econ-CB resource logic
- Enforce-OR malicious actor detection
- handleAgentDecision processor
- Audit-OR data integrity hashing

### Quick Start

2. Open `index.html`
3. Enter Groq key
4. Explore the 7 agents
